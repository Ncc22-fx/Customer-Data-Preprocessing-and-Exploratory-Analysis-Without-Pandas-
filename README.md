# Customer-Data-Preprocessing-and-Exploratory-Analysis-Without-Pandas

### Project Overview

This project focuses on cleaning, preprocessing, and analyzing customer data using core Python techniques without relying on the Pandas library for data manipulation.
The goal was to demonstrate a deeper understanding of raw data processing by handling datasets manually using Python built-in data structures such as:

Lists
Dictionaries
Loops
Conditional statements
File handling techniques

After preprocessing the dataset, exploratory data analysis and visual insights were generated using Seaborn and Matplotlib.

## Problem Statement

Customer datasets often contain:

Missing values
Duplicate records
Inconsistent formatting
Invalid entries
Unstructured information

These issues reduce data quality and affect business decision-making.

This project solves that challenge by manually preprocessing customer data before conducting meaningful analysis.

Clean raw customer data manually
